以取代和超越電子通用計算作為其終極目標，集成光學已經取得了長足的進展。硅、氮化硅、碳化硅、鈮酸鋰、氮化鋁等各種材料都得到了廣泛研究。我們在這一領域的研究重點包括以下幾個方面：

1. 將新穎的光學現象和材料引入集成光子學。
2. 探討在（大規模）光子集成前的必要和關鍵條件。
3. 將最先進的微電子技術引入片上光子集成。
4. 實現那些具有巨大潛力但被長期忽視的理論研究。

第一點的一個例子是：我們將新穎的*暗脈衝列的時間Talbot效應* {% cite Wu2022a %} 引入了集成光學領域 {% cite Wu2023 %}。

第二個方面的一個例子是增強了近零折射率材料中的熱光效應 {% cite Wu2024a %}，這些材料也與CMOS兼容（參見[近零折射率非線性納米光子學項目]({% link _projects/zh-hk/proj_enz_photonics.md %}))。
引入最先進的微電子技術可以帶來意想不到的有趣新發現，例如，我們首次將超臨界流體技術應用於簡併半導體光子材料 {% cite Wu2021b %}。
我們還旨在將早期的理論研究（參見[科研成果](/publications/)）應用於集成納米器件（納米結構、薄膜器件、超表面）、芯片（混合與新型材料）、電路（具有特定拓撲的網絡），並結合我們新開發的技術，與全球的合作實驗室和同事共同開展研究。
