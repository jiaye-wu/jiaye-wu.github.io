近零介電常數光子學可以展現出很多奇特的線性與非線性光學性質 {% cite Wu2021 %}。
我們過去與未來的的研究方向可以被總結為以下四類：

1. **製備與加工**：製備方法、工序配方以及滿足特種需求（例如 {% cite Huang2023 Wu2021b%}）。
2. **光學片上集成**：先決條件研究（預研）、設計參考、混合集成（例如 {% cite Wu2024a Sha2022%}）。
3. **新穎光學現象**：新穎且充滿潛力的線性與非線性光學效應（例如 {% cite Wu2024a Wu2022 Wu2021a %}）。
4. **微納與集成器件應用**：光計算、光通信與信號處理相關（例如 {% cite Xie2020a Sha2022 Xie2022 Sha2021%}）。。
