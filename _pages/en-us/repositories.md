## useful GitHub repositories

---

### scientific tools

[MIT MEEP](https://github.com/NanoComp/meep): free finite-difference time-domain (FDTD) software for electromagnetic simulations.

[QUICKphotonics](https://github.com/jiaye-wu/QUICKphotonics): Quick UnIt Conversion Kit for photonics.

[SSPROP-NFOL](https://github.com/jiaye-wu/SSPROP-NFOL): Split-Step Fourier Transform Propagation NFOL flavor with [sample codes](https://github.com/jiaye-wu/SSPROP-NFOL-DemoCode)  for Agrawal's _Nonlinear Fibre Optics_.

[Engauge Digitizer](https://github.com/akhuettel/engauge-digitizer): get data from plots and figures.

### paper writing and LaTeX

[bibtex-tidy](https://github.com/FlamingTempura/bibtex-tidy): format your bibtex file.

[libpku](https://github.com/lib-pku/libpku): course data from Peking University.

[pkuthss](https://github.com/CasperVector/pkuthss): LaTeX template for Peking University theses.

[EPFL thesis template](https://github.com/glederrey/EPFL_thesis_template): LaTeX template for EPFL theses.

[PowerToys](https://github.com/microsoft/PowerToys): enhance your Microsoft Windows experience when doing science. useful modules include Quick Accent, Windows Explorer Add-ons, PowerRename, image resizer, FancyZones, PowerToys Run, etc.
